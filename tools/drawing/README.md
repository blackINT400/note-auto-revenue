# FX描画ツール

FXのチャート分析用に、波形・矢印・図形を自由に描き込める描画ツールです。用途に応じて2種類あります（どちらも**費用ゼロ**）。

| ツール | ファイル | 用途 |
|---|---|---|
| **オーバーレイ版 (Mac)** | `mt4_overlay_mac.py` | **MacでMT4の上に直接描く**。トレード中のライブ注釈向け |
| オーバーレイ版 (Windows) | `mt4_overlay.py` | WindowsでMT4の上に直接描く |
| ブラウザ版 | `index.html` | チャートのスクショを貼り付けて描く。プラン作成・記録・共有向け |

> オーバーレイ版はOSごとに仕組みが違うため、Mac用(`mt4_overlay_mac.py`)とWindows用(`mt4_overlay.py`)を分けています。

---

## ⓪ オーバーレイ版（Macで MT4 の上に直接描く）— `mt4_overlay_mac.py`

macOS の Cocoa(pyobjc) を使った、常時最前面の透明オーバーレイです。追加インストールは `pyobjc` 1つだけ（費用ゼロ）。

### 使い方
1. ターミナルで初回だけ依存を入れる:
   ```bash
   pip install pyobjc-framework-Cocoa
   ```
2. 起動:
   ```bash
   python mt4_overlay_mac.py
   ```
   （`run_mac.command` をダブルクリックすると、依存の導入と起動を自動で行います）
3. MT4を開いた状態で、上部ツールバーの **「描画中」** で描く → トレード時は押して **「操作中」** に切替

### モード
起動直後は **「操作モード」**（マウスは素通り＝閉じ込められない安全状態）で始まります。描くときだけ「描画」ボタンか `F8` で描画モードに切り替えます。

| モード | 状態 |
|---|---|
| **操作中** | クリックがMT4に**通り抜ける**。描いた線は残る（起動時はこちら） |
| **描画中** | MT4の上に自由に描ける |

### 終了方法（どれでも／必ず抜けられます）
| 方法 | 動作 |
|---|---|
| `Esc` | 終了 |
| `Cmd+Q` | 終了 |
| `✕` ボタン | 終了 |
| `Ctrl+C` | 起動元の**ターミナル**で押すと強制終了 |

### ショートカット（アプリがアクティブな間＝主に描画中に有効）
| キー | 動作 |
|---|---|
| `F8` | 描画 ⇔ 操作 の切替 |
| `Cmd+Z` | 元に戻す |
| `P` / `L` / `A` / `R` / `C` | ペン / 直線 / 矢印 / 四角 / 円 |

### 保存
`💾` で画面全体を `~/Desktop` にPNG保存（macOS標準の `screencapture` を使用、追加ライブラリ不要）。

> グローバルなキー監視は使わない設計なので、**アクセシビリティ許可は不要**です。

## ① オーバーレイ版（Windowsで MT4 の上に直接描く）— `mt4_overlay.py`

MT4（や任意のアプリ）の画面の**上に重ねて**矢印・波形・図形を描ける、常時最前面の透明オーバーレイです。Python標準の `tkinter` だけで動くので追加インストール不要（Windows想定）。

### 使い方
1. Python 3 をインストール（無料 / <https://www.python.org> ）
2. `mt4_overlay.py` をダブルクリック、または `python mt4_overlay.py`
3. MT4を開いた状態で **「描画」** を押して描く → トレードしたい時は **「操作」** を押す

### モード（ここが肝）
| モード | 状態 |
|---|---|
| **描画モード** | MT4の上に自由に描ける。MT4はうっすら透けて見える |
| **操作モード** | クリックがMT4に通り抜ける。描いた線は残したままMT4を普通に操作できる |

ツールバー（画面上部・ドラッグで移動可）で モード切替 / ツール（ペン・直線・矢印・四角・円）/ 色 / 太さ / 元に戻す / 全消去 / 保存 / 終了 を操作します。

### ショートカット
| キー | 動作 |
|---|---|
| `F8` | 描画 ⇔ 操作 の切替 |
| `Ctrl+Z` | 元に戻す |
| `Delete` | 全消去 |
| `Esc` | 終了 |

### 保存について
`💾` は画面全体をPNG保存します（`Pillow` が必要: `pip install pillow`）。無くても描画・注釈自体は使えます。

> メモ: 透明・クリック透過は Windows の機能を使います。MT4はWindows想定のため最適化しています。

## ② ブラウザ版（スクショに描く）— `index.html`

**インストール・費用ゼロ**。`index.html` をブラウザで開くだけで動きます。

### 使い方

1. `tools/drawing/index.html` をダブルクリック（またはブラウザにドラッグ）して開く
2. ツールバーで道具を選んでキャンバス上をドラッグ

## 機能

| 分類 | 内容 |
|---|---|
| 描画 | ペン ✏️ / 消しゴム ⌫ |
| 図形 | 直線 ╱ / 矢印 ➜ / 四角 ▭ / 円 ◯ |
| スタイル | 色（カラーピッカー＋赤緑黄青黒のスウォッチ）/ 太さ 1〜40px |
| 編集 | 元に戻す ↶ / やり直し ↷ |
| 画像 | チャート画像の読み込み（🖼）・**Ctrl+V で貼り付け** |
| 出力 | PNG保存 💾 / 全消去 🗑 |

## キーボードショートカット

| キー | 動作 |
|---|---|
| `P` | ペン |
| `E` | 消しゴム |
| `L` | 直線 |
| `A` | 矢印 |
| `R` | 四角 |
| `C` | 円 |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` / `Ctrl+Shift+Z` | やり直し |
| `Ctrl+V` | クリップボードの画像を貼り付け |

## FXでの典型的な使い方

1. 取引ツールのチャートをスクリーンショット（Win: `Win+Shift+S`）
2. このツールで `Ctrl+V` して貼り付け
3. 矢印・水平線・波形を描き込んでトレードプランを作成
4. `💾 保存` でPNGに書き出し、記録・共有

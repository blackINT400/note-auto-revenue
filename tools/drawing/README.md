# FX描画ツール

FXのチャート分析用に、波形・矢印・図形を自由に描き込めるブラウザ完結の描画ツールです。
**インストール・費用ゼロ**。`index.html` をブラウザで開くだけで動きます。

## 使い方

1. `tools/drawing/index.html` をダブルクリック（またはブラウザにドラッグ）して開く
2. ツールバーで道具を選んでキャンバス上をドラッグ

## 機能

| 分類 | 内容 |
|---|---|
| 描画 | ペン ✏️ / 消しゴム ⌫ |
| 図形 | 直線 ╱ / 矢印 ➜ / 四角 ▭ / 円 ◯ |
| スタイル | 色（カラーピッカー＋赤緑黄青黒のスウォッチ）/ 太さ 1〜40px |
| 編集 | 元に戻す ↶ / やり直し ↷ |
| 画像 | チャート画像の読み込み（🖼）・**Ctrl+V で貼り付け** |
| 出力 | PNG保存 💾 / 全消去 🗑 |

## キーボードショートカット

| キー | 動作 |
|---|---|
| `P` | ペン |
| `E` | 消しゴム |
| `L` | 直線 |
| `A` | 矢印 |
| `R` | 四角 |
| `C` | 円 |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` / `Ctrl+Shift+Z` | やり直し |
| `Ctrl+V` | クリップボードの画像を貼り付け |

## FXでの典型的な使い方

1. 取引ツールのチャートをスクリーンショット（Win: `Win+Shift+S`）
2. このツールで `Ctrl+V` して貼り付け
3. 矢印・水平線・波形を描き込んでトレードプランを作成
4. `💾 保存` でPNGに書き出し、記録・共有
